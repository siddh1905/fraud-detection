# fraud-detection
 Fraud Detection System is a project focused on identifying suspicious or unauthorized financial transactions using a combination of rule-based methods, machine learning models, and behavioral analysis. The goal is to help banks, digital wallets, and financial platforms detect fraud early, reduce losses, and ensure secure transactions.

Fraud detection is complex, involving AI, machine learning, and behavioral biometrics. This project translates those technical concepts into an **interactive web experience**. 
Instead of reading a boring PDF, users can explore data visualizations, walk through the detection pipeline step-by-step, and even play the role of a bank analyst to spot fraud in a gamified simulator.

* **Interactive Dashboard:** Visualizes attack vectors and risk factors using dynamic charts (Chart.js).

* **Analyst Simulator:** A mini-game where users judge real-time transactions based on risk signals (IP, velocity, device).
* **Comparison Engine:** Visualizes the difference between legacy "Rules-based" systems and modern "AI" detection.
* **Downloadable:** Users can save the entire report as a PDF or HTML file for offline use.
